The project using as accelerometer to take in the tilt from the default position translating this to a numerical value also input from 2 buttons and a joystick (for scroll). this data is transmitted over a wireless channel (433Mhz) to a receiver which converts this data to computer understandable mouse codes over PS/2 interface
